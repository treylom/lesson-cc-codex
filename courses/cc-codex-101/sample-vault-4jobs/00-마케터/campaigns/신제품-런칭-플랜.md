# 신제품 런칭 플랜 (체크리스트)

신제품 낼 때마다 빠뜨리는 게 생겨서 만든 표준 체크리스트.
지금은 [[여름-신제품-캠페인-초안]] 접이식 빨래바구니 기준으로 채우는 중.

## D-21 (3주 전)
- [ ] 제품 최종 스펙·컬러·가격 확정
- [ ] 상세페이지 기획 시작
- [ ] 메인 비주얼 촬영 일정 잡기 ([[비주얼-가이드]] 기준)

## D-14 (2주 전)
- [ ] 상세페이지 1차 완성
- [ ] 인스타 릴스 촬영 ([[인스타-릴스-스크립트]])
- [ ] 뉴스레터 초안 ([[뉴스레터-초안-6월]])
- [ ] 블로그 정보글 예약 ([[블로그-초안-자취살림]])

## D-7 (1주 전)
- [ ] 구매고객 예고 메일 발송
- [ ] 광고 소재 점검 — '생활감 버전' 꼭 포함 (봄 캠페인 교훈)
- [ ] 재고 확인 — 인기 예상 컬러 넉넉히

## D-DAY
- [ ] 상품 등록·노출 확인
- [ ] 첫날 프로모션 코드 적용 확인
- [ ] 인스타 피드 게시

## D+14 (런칭 2주 후)
- [ ] 재구매 유도 메일 (구매 후 2~3주가 타이밍 — [[봄시즌-캠페인-회고]])
- [ ] 초기 성과 점검 ([[채널별-성과-메모]] 업데이트)

## 자주 빠뜨리는 것
- 재고 확인을 막판에 함 → 미리.
- 광고가 너무 깔끔하게만 나옴 → 생활감 컷 필수.
