# 고객 페르소나

콘텐츠 쓸 때 "이 사람한테 말한다" 생각하면서 쓰려고 정리한 가상 인물 셋.

## 1. 자취 5년차 김지윤 (32세, 직장인)
- 서울 원룸 거주, 출퇴근 왕복 2시간. 평일엔 집안일 거의 못 함.
- 주말 몰아서 청소·빨래. "최소한의 노력으로 깔끔하게"가 목표.
- 인스타에서 살림템 자주 봄. 비싸도 오래 쓰면 산다. 충동구매는 잘 안 함.
- 우리한테: 좁은 공간 효율, 손 덜 가는 제품 어필이 통함.

## 2. 신혼 2년차 박서준·이하늘 부부 (35세 / 33세)
- 신축 아파트 신혼집. 집 꾸미기에 관심 많고 통일감 중시.
- 둘 다 맞벌이라 살림은 나눠서. 디자인이랑 실용성 둘 다 봄.
- 정보 탐색을 꼼꼼히 함. 리뷰 많이 읽고 비교 후 구매.
- 우리한테: 컬러·디자인 통일성, 세트 구성 어필이 통함.

## 3. 워킹맘 최은영 (41세, 초등 자녀 둘)
- 시간이 늘 부족. "한 번에 여러 개 해결되는" 제품 선호.
- 가격에 민감하지만 아이 관련은 품질 우선.
- 네이버 카페·블로그에서 정보 얻음. 입소문 중요.
- 우리한테: 시간 절약, 대용량·실속 구성, 안전한 소재 어필이 통함.
