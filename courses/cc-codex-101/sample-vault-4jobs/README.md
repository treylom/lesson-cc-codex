# 직무별 샘플 vault (마케터 · 기획 · 연구 · 운영)

패캠 강의 **「클로드 코드 & Codex로 만드는 나만의 AI 업무 비서」** 실습용 샘플 보관함(vault)입니다.

## 이게 뭔가요?
직무마다 "내 업무 자료를 옵시디언에 이렇게 모아두면 된다"를 보여주는 **예시 모음**입니다. 네 직무(마케터·기획·연구·운영) 각각의 폴더 안에, 그 직무가 실제로 다룰 법한 자료(브랜드 가이드·회의록·논문 요약·주간 보고 등)를 샘플로 담았어요.

## 어떻게 쓰나요?
- **옵시디언에서 이 폴더(`sample-vault-4jobs`)를 "보관함으로 열기"** 하면 그대로 둘러볼 수 있습니다.
- 또는 강의를 따라가며, 내 직무 폴더를 골라 그 구조를 내 자료로 바꿔보세요.
- 여기 자료는 **예시**입니다 — 실제 회사·제품이 아니라 일반적인 상황으로 꾸몄어요. 마음껏 고치고 지워도 됩니다.

## 폴더 구성
| 폴더 | 직무 | 핵심 장면 |
|---|---|---|
| `00-마케터/` | 마케터 | 흩어진 자료(브랜드·캠페인·고객·경쟁사)를 한곳에 → 톤 일관된 콘텐츠 |
| `01-기획/` | 기획자 | 회의록·리서치 → 제안서 초안 / **결정의 '이유'를 남긴다** |
| `02-연구/` | 연구자 | 논문·자료를 **위키링크로 연결**해 지식 연결망으로 키운다 |
| `03-운영/` | 운영자 | 팀의 장기 기억 → 주간 보고·온보딩 자동 초안 |

각 폴더 안의 **`CLAUDE.md`** = AI에게 "이 폴더는 무슨 자료고, 정리할 땐 이렇게 해줘"라고 일러두는 한두 줄짜리 메모입니다(강의에서 배우는 "규칙서"의 작은 예).

## 공통 골격 (어느 직무든 똑같아요)
**숫자 붙인 폴더 + `CLAUDE.md` 한 줄 + 자료 넣기.** 직무마다 바뀌는 건 ① 폴더 이름 ② 담는 자료 ③ 자주 던지는 질문 — 이 세 가지뿐. "완벽한 폴더 구조"를 고민할 필요 없이, 내 직무 버전만 고르면 됩니다.

> 강의 클립 2.2.3에서 이 vault를 직접 열어 보여드립니다. Part 3에서는 같은 vault로 knowledge-manager 실습을 이어 갑니다.
