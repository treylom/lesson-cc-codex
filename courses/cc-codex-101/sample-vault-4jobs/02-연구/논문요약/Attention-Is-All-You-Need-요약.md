# Attention Is All You Need 요약

- **저자**: Vaswani 외 7명 (Google)
- **발표**: 2017년, NeurIPS
- **원문**: https://arxiv.org/abs/1706.03762 (접근일 2026-03-12)

## 한 줄 요약
문장을 처리할 때 기존처럼 단어를 하나씩 순서대로 읽지 않고, **모든 단어를 한 번에 보면서 서로의 관계를 따지는** 새로운 구조(트랜스포머)를 제안한 논문. 지금 나오는 거의 모든 LLM의 출발점이다.

## 왜 중요한가
- 그전까지는 RNN/LSTM처럼 단어를 앞에서 뒤로 순서대로 읽었다. 문장이 길어지면 앞부분을 잊어버리고, 순서대로 처리하니 느렸다.
- 이 논문은 "순서대로 읽을 필요 없이 어텐션(attention, 어떤 단어가 어떤 단어를 주목할지 계산) 만으로 충분하다"고 주장했다. 그래서 제목이 "Attention Is All You Need".

## 핵심 아이디어 3가지
1. **셀프 어텐션** — 한 문장 안에서 단어들끼리 서로 얼마나 관련 있는지 점수를 매긴다. "그것은"이 가리키는 게 뭔지 같은 걸 잡아낸다.
2. **병렬 처리** — 모든 단어를 동시에 계산하니 학습이 훨씬 빠르다. GPU를 제대로 쓸 수 있게 됐다.
3. **위치 정보 추가** — 순서대로 안 읽으니 단어 위치를 따로 표시해준다(positional encoding).

## 한계 / 메모
- 문장이 아주 길어지면 계산량이 제곱으로 늘어 비용이 큼 (이후 논문들이 이 문제를 개선).
