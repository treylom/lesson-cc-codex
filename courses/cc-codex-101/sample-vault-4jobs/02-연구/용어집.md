# 용어집

자료 읽다 막히는 용어를 한 줄씩 모아두는 곳. 비전공자 기준으로 쉽게. 깊은 설명은 옆 링크로.

## 기본
- **LLM (대규모 언어 모델)** — 엄청난 양의 글을 학습해 다음에 올 말을 잘 맞히는 AI. ChatGPT 같은 것의 속.
- **토큰(token)** — AI가 글을 다루는 최소 단위. 대략 단어 한두 개. 비용·길이 제한이 토큰 기준.
- **컨텍스트 윈도우** — AI가 한 번에 볼 수 있는 글 분량. 넘치면 앞부분이 잘림. → [[LLM-에이전트-메모리-정리]]

## 구조 / 동작
- **트랜스포머** — 요즘 LLM의 기본 골격. → [[트랜스포머-한장정리]]
- **어텐션** — "어디를 더 봐야 하나"를 정하는 부품. → [[어텐션-쉽게]]
- **임베딩** — 뜻을 숫자 좌표로 바꾼 것. → [[임베딩-한장정리]]

## 활용
- **RAG (검색 증강 생성)** — 답하기 전에 관련 자료를 찾아 붙이는 방식. → [[RAG란-무엇인가]]
- **파인튜닝** — 모델을 우리 자료로 추가 학습. → [[파인튜닝-vs-RAG]]
- **프롬프트** — AI에게 주는 지시문. 잘 쓰는 법 → [[프롬프트-잘-쓰는-법]]
- **에이전트** — (정의가 글마다 다름) 좁게는 "스스로 계획을 세우고 도구를 써서 일을 끝내는 AI". 넓게는 단순 자동화까지 포함하기도. 헷갈리면 글쓴이가 어느 뜻으로 쓰는지 먼저 확인.
- **할루시네이션(환각)** — AI가 모르는 걸 그럴듯하게 지어내는 것. RAG가 줄여준다.

## 메모
- 새 용어 만나면 여기 한 줄 추가하고, 깊으면 개념 노트로 빼서 링크 걸기.
