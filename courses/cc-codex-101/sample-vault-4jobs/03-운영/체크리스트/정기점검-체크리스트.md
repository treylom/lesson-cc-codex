# 정기 점검 체크리스트

매주 월요일 오전에 한 바퀴 돕니다. 큰일이 터지기 전에 미리 잡는 게 목적이에요. 10~15분이면 끝납니다.

## 서버 / 저장공간
- [ ] 서버 디스크 여유 공간 확인 (20% 밑이면 정리)
- [ ] 오래된 로그 파일 정리 (한 달 지난 건 압축 또는 삭제)
- [ ] 백업이 지난주에 정상적으로 돌았는지 확인

## 모니터링 / 알림
- [ ] 대시보드 에러율이 평소 범위인지
- [ ] 지난주 알림 중 오탐이 몇 건이었는지 (많으면 임계치 조정 검토)

## 고객 / 처리량
- [ ] 미처리 고객문의가 쌓여 있지 않은지 (3일 넘은 티켓 확인)
- [ ] SLA(첫 응답 6시간) 지난주 평균 점검

## 일정 / 사람
- [ ] 이번 주 배포 예정 건 확인
- [ ] 온콜 당번이 정해져 있는지 ([[온콜-당번-안내]])
- [ ] 휴가·외근으로 자리 비는 사람 파악

> 점검 결과는 [[주간운영보고-2026-W25]] "이번 주 한 일"에 한 줄로 남깁니다. 이상 없으면 "정기 점검 이상 없음"이라고만 적어도 됩니다.
