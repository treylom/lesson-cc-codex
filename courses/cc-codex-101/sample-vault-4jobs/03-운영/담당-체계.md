# 누가 뭘 맡나 (담당 체계)

"이건 누구한테 물어봐야 하지?" 싶을 때 보는 표입니다. 사람이 바뀌면 여기부터 고칩니다.

## 역할 분담
| 영역 | 담당 | 백업(자리 비울 때) |
|---|---|---|
| 배포 / 릴리즈 | 김민수 | 이상훈 |
| 모니터링 / 장애 1차 | 온콜 당번 ([[온콜-당번-안내]]) | 팀장 |
| 고객문의 처리 | 박지영 + 김민수 | — |
| 협력사 정산 / 마감 | 이상훈 | 김민수 |
| 신입 온보딩 | 사수(그때그때 지정) | — |

## 이럴 땐 이 사람
- 배포 일정 잡고 싶다 → 김민수
- 새벽에 알림이 울렸다 → 그 주 온콜 당번
- 정산·돈 관련 → 이상훈
- 권한·계정 문제 → IT에 [[업무요청-받는법]] 양식으로

## 우리 팀 밖
- 계정·서버 권한: IT팀
- 채용·휴가: 인사팀
- 고객 영향 큰 공지: 영업팀과 함께

> 담당자가 휴가면 백업이 대신 봅니다. 백업도 비면 팀장에게. 혼자 끙끙대지 않기는 [[팀-핸드북]] 규칙입니다.
