# 신입 첫 주 가이드

환영합니다! 첫 주는 적응이 목표예요. 다 못 외워도 괜찮으니 어디에 뭐가 있는지만 익혀두세요.

## 첫날
- [ ] 계정 발급 확인 (메신저, 대시보드, 티켓 시스템, 옵시디언)
- [ ] 사수와 1:1 인사 미팅 (30분)
- [ ] [[팀-핸드북]] 한 번 정독
- [ ] 팀 채널 `#운영-알림`, `#배포공지` 들어가기

## 2~3일차
- [ ] 모니터링 대시보드 둘러보기 (서버 상태 어디서 보는지)
- [ ] 사수 옆에서 고객문의 처리 과정 어깨너머로 보기
- [ ] [[신규기능-배포-체크리스트]] 읽고 왜 이런 항목이 있는지 사수에게 질문

## 4~5일차
- [ ] 간단한 고객문의 1~2건 직접 처리 (사수 확인 받고 답변)
- [ ] 주간보고가 어떻게 작성되는지 [[주간운영보고-2026-W25]]로 형식 익히기
- [ ] 모르는 용어 정리해서 사수에게 한 번에 물어보기

## 지난 신입들이 막혔던 지점 (미리 알아두세요)
- **계정 발급이 늦어요**: 입사 당일 계정이 안 나오는 경우가 잦았습니다. 안 나왔으면 바로 사수에게 말하세요. 기다리지 말고요.
- **용어가 낯설어요**: '핸드오프', '온콜', '롤백' 같은 말이 처음엔 어렵습니다. 핸드북 용어를 그때그때 찾아보면 됩니다. 모아서 물어봐도 좋아요.
- **혼자 끙끙대다 시간 버려요**: 30분 넘게 막히면 무조건 물어보기. 이게 우리 팀 규칙입니다.
- **배포가 무서워요**: 처음엔 다 그렇습니다. 체크리스트대로만 하면 됩니다. 둘째 주에 사수와 같이 한 번 해볼 거예요.

## 첫 주 끝나면
사수와 회고 미팅 — 뭐가 어려웠는지, 뭘 더 알고 싶은지 편하게 이야기하면 됩니다.
