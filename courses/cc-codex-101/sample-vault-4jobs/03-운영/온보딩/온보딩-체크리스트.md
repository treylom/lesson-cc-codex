# 온보딩 체크리스트 (사수용)

[[신입-첫주-가이드]]가 신입 본인이 보는 안내라면, 이건 **사수가 빠뜨리지 않으려고 체크하는 목록**입니다. 신입 합류 결정되면 사수가 이 목록을 복사해서 씁니다.

## 입사 1주 전 (미리 준비)
- [ ] IT에 계정 발급 요청 (메신저·대시보드·티켓·옵시디언) — **한 주 전 미리!** ([[업무요청-받는법]] 양식)
- [ ] 자리·장비 준비 확인
- [ ] 첫 주 일정 잡아두기 (인사 미팅, 1:1 시간)

## 입사 첫날
- [ ] 모든 계정 실제 로그인되는지 같이 확인
- [ ] 팀 채널 초대 (`#운영-알림`, `#배포공지`)
- [ ] [[팀-핸드북]] 어디 있는지 안내
- [ ] 점심 같이 (분위기 풀어주기)

## 첫 주 안에
- [ ] [[신규기능-배포-체크리스트]] 같이 읽기
- [ ] 고객문의 처리 어깨너머로 보여주기
- [ ] 모르는 용어 모아 물어볼 수 있게 안내

## 첫 주 끝
- [ ] 회고 미팅 (뭐가 어려웠는지)
- [ ] 둘째 주 계획 합의 (배포 한 번 같이 해보기 등)

> 매번 걸리는 게 계정 발급입니다. **입사 당일에 요청하면 늦습니다. 무조건 한 주 전.** 이 한 줄만 지켜도 신입 첫날이 편해집니다.
