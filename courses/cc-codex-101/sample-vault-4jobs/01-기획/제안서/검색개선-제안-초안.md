# 검색 기능 개선 제안 (초안)

> 작성 중. 기대효과 수치는 정해린님 데이터 받고 채울 것. → [[리서치-메모]]

## 1. 배경
우리 서비스 검색은 출시 이후 거의 손대지 않았다. 그동안 상품 수가 3배 가까이 늘면서, 사용자가 원하는 걸 찾기가 점점 어려워지고 있다.

## 2. 문제
- 검색은 단어가 정확히 일치해야만 결과가 나온다. 띄어쓰기나 오타가 있으면 빈 화면.
- 두 단어 이상 입력하면 결과가 거의 안 나옴. (6/15 회의 기준 검색 후 이탈률 41%) → [[2026-06-15-신규기능-기획회의]]
- CS 문의 중 검색 관련 불만이 약 23%로 가장 많다.

## 3. 제안
- **오타 보정**: 자주 틀리는 입력을 보정해서 검색 (예: "노트북 가방" → "노트북가방")
- **부분일치**: 입력한 단어가 포함되기만 해도 결과에 노출
- **결과 없음 화면 개선**: 빈 화면 대신 "이런 걸 찾으셨나요?" 추천 노출

## 4. 기대효과
- 검색 후 이탈률 41% → (목표 30% 이하)
- 검색 관련 CS 문의 감소
- (구체 수치는 데이터 확인 후 보완 예정)

## 5. 일정(안)
- 6월 말: 부분일치 우선 적용
- 7월: 오타 보정 + 결과 없음 화면

## 남은 질문
- 부분일치하면 서버 부하가 얼마나 늘까? (박민호님 확인 중)
- 추천 화면에 뭘 보여줄지 기준이 필요함
