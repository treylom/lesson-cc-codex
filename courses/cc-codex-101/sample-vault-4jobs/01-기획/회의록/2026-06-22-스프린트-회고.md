# 6월 3주차 스프린트 회고

- 일시: 2026년 6월 22일 (월) 16:00~16:50
- 참석자: 김선영(기획), 박민호(개발), 이지원(디자인), 정해린(데이터)

이번 스프린트(6/8~6/22)는 검색 개선 준비에 집중했다. 잘된 점·아쉬운 점·다음에 바꿀 점을 가볍게 나눔.

## 잘된 것 (Keep)
- 검색 실패 키워드 상위 50개 정리가 빠르게 나왔다. 덕분에 회의가 구체적이었다. → [[2026-06-15-신규기능-기획회의]]
- 인터뷰를 짧게라도 돌린 게 결정에 큰 도움. → [[리서치-메모]]

## 아쉬운 것 (Problem)
- 부분일치 적용 시 서버 부하 예상치가 아직 안 나왔다. 박민호님이 다른 일에 밀림.
- 제안서 기대효과 수치가 비어 있어서 윗선 보고가 미뤄졌다. → [[검색개선-제안-초안]]

## 다음에 바꿀 것 (Try)
- 외부 의존(데이터·부하 측정) 작업은 스프린트 시작할 때 먼저 요청해두기. 막판에 몰리면 항상 늦는다.
- 제안서는 "수치 빈칸"이라도 일단 공유해서 피드백을 먼저 받자.
