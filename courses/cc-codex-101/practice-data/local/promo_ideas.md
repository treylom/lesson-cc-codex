> 사장 지우가 틈틈이 적어둔 아이디어 메모. 아직 정리 안 된 초안 상태다.

## 지난 5월에 해본 것 (효과 메모)

- **인스타 해시태그 인증 이벤트** (5/22~5/24): 인증하면 아메리카노 사이즈업.
  - 체감: 5/22~24 방문객·신규 손님 늘어난 느낌. 매출표로 확인 필요.
  - 비용 거의 안 듦. 다만 사이즈업 원가 부담은 작게 있음.
- **주말 원데이 클래스 정례화**: 토요일마다 캔들/라떼아트/드립백 번갈아.
  - 클래스 자체 매출 + 클래스 손님이 음료·디저트도 같이 사감. 효과 좋아 보임.

## 떠오른 아이디어 (검증 전)

1. **평일 낮 한산 시간(14~17시) 할인** — 리뷰에서 "평일 낮 할인 있으면 자주 오겠다" 반복.
   - 고민: 할인하면 마진이 주는데, 원래 한가한 시간이라 추가 손님이면 이득일 수도. → 평일 낮 매출 비중부터 보자.
2. **레모네이드 병 판매(테이크홈)** — "병으로 팔면 산다" 후기 여러 건.
   - 수제청 재고·위생 용기 발주 필요. 소량 시범 판매부터?
3. **클래스 재료 키트 단품 판매** — 캔들 키트 사고 싶다는 DM 반복.
   - 발주처(공방마켓)에서 키트만 추가 주문 가능한지 확인.
4. **인기 디저트 예약제** — 무화과타르트·스콘 오후 품절 불만 많음.
   - 전날 예약받아 수량 맞추면 폐기도 줄고 헛걸음도 줄어듦.
5. **클래스 노쇼 방지** — 노쇼로 빈자리 아까움.
   - 예약금 일부 선결제? 전날 리마인드 문자? 둘 다?
6. **비 오는 날 영업/메뉴 안내** — "비 오면 여나요?" 문의 반복.
   - 비 오는 날 매출이 확 떨어짐(매출표 확인). 비 오는 날 한정 "우산 손님 디저트 +1" 같은 거 어떨까.

## 결정할 때 같이 볼 것 (사장 메모)

- 어떤 아이디어가 **돈이 적게 들고 효과가 빠른가**?
- 매출표에서 **약한 구간(비 오는 날·평일 낮)**을 메우는 게 우선일까, **잘 되는 구간(주말·클래스)**을 키우는 게 우선일까?
- 리뷰에서 **반복**되는 요청부터 처리하면 손님 만족도와 직결됨.
