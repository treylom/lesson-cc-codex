# practice-data 회의록 모음 (cc-codex-101 실습용)

## 회의 1 — 2025-04-10 운영 주간 회의
- 온라인 채널 매출이 3월 들어 뚜렷이 늘었으나, 반품도 같이 증가했다.
- 운영팀은 "신규 프로모션으로 주문은 늘었는데, 배송 지연 때문에 반품 문의가 따라 늘었다"고 보고.
- 파트너 채널은 신규 제휴처 교육이 늦어 매출 기여가 기대보다 약했다.
- 다음 분기엔 채널별 배송 SLA(약속 배송일) 점검을 우선하기로 했다.

## 회의 2 — 2025-07-15 고객지원 합동 회의
- 7~8월 고객지원 문의(support_tickets)가 다른 달보다 급증했다.
- 지원팀은 "여름 주문 폭증 + 배송 지연이 겹쳐 반품·문의가 동시에 튀었다"고 설명.
- 반복 문의 1위 = 배송 추적, 2위 = 반품 절차 안내.
- 운영팀은 FAQ 문서 정비와 자동 안내가 문의량을 줄일 것이라 판단.
- 3분기엔 상위 문의 유형을 매출·반품 데이터와 함께 추적하기로 했다.

## 회의 3 — 2025-11-20 분기 리뷰 회의
- 4분기 들어 온라인·파트너 채널 모두 빠르게 성장(연말 시즌).
- 다만 12월 반품·문의가 다시 오르는 조짐 → 시즌 물량 대비 지원 인력 부족 우려.
- 파트너 채널은 초기 교육이 충분했던 제휴처일수록 성과가 좋았다.
- 팀장: "내년엔 AI로 매출·반품·문의 데이터를 묶어 한눈에 보는 보고 체계를 만들자."
- 액션: 채널별 성공 사례 + 운영 개선 포인트를 연말 보고서에 함께 담기.
