# 제안서·포트폴리오 메모 — 지우 (1인 컨설턴트/프리랜서)

혼자 일하면서 제안서를 쓸 때마다 매번 새로 만들면 시간이 너무 든다. 그래서 자주 쓰는 단가·조건·표현을 여기 정리해 둔다. 새 견적을 낼 때 이 메모를 AI에게 같이 읽히면 "내 기준에 맞는" 제안서 초안을 빨리 받을 수 있다.

---

## 1. 작업 유형별 기본 단가 (2026년 상반기 기준)

| 작업 유형 | 기본 단가 | 비고 |
|----------|----------|------|
| 로고 디자인 | 80만~150만원 | 시안 3종 + 수정 3라운드 포함 |
| 상세페이지(1종) | 35만~50만원 | 사진 보정은 별도 단가 |
| 인스타 콘텐츠(월 12종) | 180만원/월 | 월정액, 3개월 이상 권장 |
| 랜딩페이지 제작 | 250만~400만원 | 결제 연동 시 +50만원 |
| 브랜드 가이드북 | 350만~450만원 | 인쇄용 시안 포함 |
| 홈페이지 리뉴얼 | 400만~600만원 | 개발 외주 별도 협의 |
| 썸네일(대량 30종+) | 종당 4만~5만원 | 30종 이상 대량 단가 |

---

## 2. 견적에 꼭 넣는 조건 (반복 분쟁 방지용)

- **수정 라운드 명시**: 기본 견적에는 수정 3라운드까지 포함. 그 이상은 라운드당 추가비(보통 견적의 10%).
- **추가 작업 단가**: 상세페이지·배너 1종 추가 시 정해진 단가표대로. "하는 김에"는 사양한다고 미리 적어둔다.
- **결제 조건**: 100만원 이상 건은 착수금 50% 선입금. 300만원 이상은 세금계산서 발행.
- **사진/촬영**: 제품 사진 보정은 디자인과 별개. 촬영 자체는 외주 연결만 하고 직접 안 함.
- **납품물 형식**: 최종 파일 형식(AI/PSD/PNG/PDF)과 인쇄 규격(CMYK 여부)을 견적 단계에서 확정.

---

## 3. 실제로 자주 터지는 문제 (메모)

1. **수정 무한루프**: 수정 라운드를 안 정하면 5~6번까지 끌려간다. (P009 모모펫, P004 코코공방 사례 — 둘 다 추가비 받기 전까지 일정 밀림)
2. **잔금 지연**: 큰 건(가이드북·홈페이지)일수록 잔금이 늦다. 담당자 변경·회계처리 핑계가 많다. (P006 더브릿지, P011 한결법률 — 둘 다 잔금 미수금 상태)
3. **추가비 협의 타이밍**: 일이 늘어난 걸 그때그때 말 안 하면 나중에 청구가 어색해진다. 요청이 정해진 범위를 넘는 순간 바로 "이건 추가 작업입니다"라고 가볍게 알려야 한다.

---

## 4. 포트폴리오로 쓸 만한 대표 사례

- **라온카페(P001→P007)**: 로고로 시작해 배달앱 배너까지 재의뢰로 이어진 케이스. "한 번 잘하면 다시 부른다"의 모범.
- **블룸스튜디오(P002→P010→P015)**: 월정액 콘텐츠 정기 계약. 매달 안정적 수입원이 되는 구조.
- **코코공방(P004→P012)**: 상세페이지 → 브랜드 패키지로 작업 범위가 커진 케이스.
- **그린푸드(P005→P013)**: 대량 썸네일 단가로 효율 좋은 반복 거래.

> 재의뢰가 들어온 고객은 별표를 쳐둔다. 신규 영업보다 재의뢰가 훨씬 남는다.

---

## 5. 제안서 톤 (내 스타일)

- 전문용어 최소화. 고객 대부분이 비전문가(사장님·1인 운영자).
- "무엇을 해드린다 → 며칠 걸린다 → 얼마다 → 이런 조건이다" 4단으로 짧게.
- 가격은 숨기지 않고 표로 명확히. 애매하게 적으면 나중에 더 피곤하다.
