# 리서치 메모 — AI 도구·1인 사업 시장 (지우)

> 강의 기획·콘텐츠 소재용으로 모아둔 리서치 메모. 출처는 단순 참고용 — 전부 실습용 가상 샘플.

---

## 1. "비개발자의 AI 코딩 도구" 시장 메모
- 클로드코드(Claude Code), Codex 같은 터미널 기반 AI 도구가 2025년 후반부터 비개발자에게도 번지는 중.
- 핵심 진입장벽 = **터미널 공포**. "검은 화면에 글자 치는 게 무섭다"는 반응이 압도적.
  - → 강의 초반 30분을 "맨바닥 새 컴퓨터에 설치부터"로 잡은 게 주효했음 (수강 완주율 상승).
- 경쟁 강의 대부분이 개발자 대상 → "비개발자·1인 사업자" 포지션이 비어 있음 = 우리 기회.

## 2. 도구 비교 메모 (강의 소재)
- 두 도구 모두 "AI 짝꿍"이라는 점은 같고, 만든 회사·기본 모델·세팅 파일 이름만 다르다.
  - 클로드코드: 규칙 파일 = `CLAUDE.md`
  - Codex: 규칙 파일 = `AGENTS.md`
- 학습자에게는 "하나를 제대로 익히면 다른 하나는 금방"이라고 안내하는 게 부담을 줄여줌.
- effort(공들이는 정도)·모델 선택으로 응답 품질이 달라진다 → 강의에서 직접 바꿔보게 하면 "와우 모먼트"가 큼.

## 3. 구독자·수강생 인터뷰에서 반복된 말 (5명)
1. "AI한테 뭘 시켜야 할지를 모르겠어요" — 활용 사례 부족이 진짜 장벽.
2. "강의는 많은데 내 업무에 어떻게 쓰는지는 안 알려줘요" — 실무 맥락 결합 수요.
3. "혼자 하니까 막히면 물어볼 데가 없어요" — 커뮤니티·Q&A 수요.
4. "엑셀은 하는데 코드는 무서워요" — 엑셀 비유로 설명하면 잘 통함.
5. "한 번에 다 알려주면 머리 아파요" — 한 번에 하나씩 원칙 재확인.

## 4. 가격·수익 구조 메모
- 객단가가 가장 높은 건 "1인 지식사업 수익화 로드맵"(149,000원)이지만, 수강 수는 "챗GPT로 콘텐츠 10배"(79,000원)가 압도적.
- 매출 기여 1위 = AI활용 카테고리. 비즈니스 카테고리는 객단가로 보완.
- 신규 "클로드코드" 강의(119,000원)는 출시 두 달 만에 단일 강의 매출 상위권 진입 → 라인업 확장 신호.

## 5. 다음에 더 찾아볼 것
- 비개발자가 실제로 자동화한 업무 사례 모음 (강의 part 후반 소재).
- 뉴스레터 → 강의 전환 퍼널 벤치마크 (전환율 평균이 궁금).
- 1인 사업자 세금·정산 자동화 도구 (수강생 문의 잦음).
