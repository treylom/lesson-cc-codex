# fc-01 (강의 클립 1.2.2 / 1.2.3): 설치와 첫 실행 (맨바닥 터미널)
> 대응 자습 레슨: `/lesson-cc-codex cc-01` · 강의 클립 1.2.2 / 1.2.3 · 예상 12~15분
> 이 문서 = 강의 촬영·진행용 런시트(이론 짧게 → 터미널 라이브). 깊은 자습은 위 레슨으로.

## 🎬 이 클립 한 줄 목표
아무것도 없는 컴퓨터에서 Claude Code를 설치·로그인하고, 한국어로 내 실제 파일의 숫자를 꺼내오는 첫 성공을 만든다.

## 🗣️ 이론 (강사 설명 · 2~4분)
- **검은 화면(터미널)은 카카오톡 채팅창의 글자 버전이다.** 잘못 쳐도 컴퓨터는 안 망가진다. 대부분 "그런 명령 없는데요?"라는 안내만 뜨고 끝.
- **Claude Code는 "내 폴더 안으로 직접 출근한 비서"다.** 채팅창 AI는 파일을 하나씩 끌어다 올려야 하지만, Claude Code는 내 폴더를 직접 열어 실제 숫자·내용을 읽고 답한다. 복사-붙여넣기 왕복 없음.
- **입력은 딱 두 종류뿐.** ① 터미널 명령어(`claude` 등) = AI를 켜고 끄는 명령, `$` 프롬프트에서 입력. ② 슬래시 명령(`/exit` 등) = AI를 켠 상태에서 채팅창 안에서 입력. 이 구분 하나만 알면 절반은 끝.
- **설치·로그인은 이 코스 전체에서 딱 한 번이다.** 내일부터는 `cd 폴더 → claude` 두 줄이면 바로 일할 수 있다.
- **한국어로 말해도 된다, 다시 시켜도 된다.** AI는 시험 감독이 아니라 짝꿍이다.

## ⌨️ 터미널 라이브 시연 (8~10분)

---

**[스텝 1] Node.js(엔진) 확인 → 설치**
- **시연**: `node --version`
- **보여줄 것**: `v20.11.0` 같은 버전 숫자가 뜨면 이미 있음. `command not found`가 뜨면 nodejs.org에서 LTS 버전 다운로드·설치 후 터미널을 껐다 다시 켜고 재확인.
- **학습자에게**: "직접 `node --version` 쳐보고 숫자가 뜨는지 확인해보세요"

---

**[스텝 2] Claude Code 설치**
- **시연**: `npm install -g @anthropic-ai/claude-code`
- **보여줄 것**: 글자가 주르륵 흐르며 1~2분 진행. 노란 `warning` 줄은 정상. 빨간 `error`로 멈추지만 않으면 성공. 완료 후 `$` 커서로 복귀.
- **학습자에게**: "직접 이 명령 복사해서 붙여넣고 실행해보세요. 창은 절대 닫지 마세요"

---

**[스텝 3] 설치 확인 + 로그인**
- **시연**: `claude --version` → 버전 숫자 확인 후 → `claude`
- **보여줄 것**: `claude --version`에서 버전 숫자(예: `2.1.183 (Claude Code)`)가 뜨면 설치 성공. 이어 `claude`를 치면 브라우저가 열리며 로그인 안내. 브라우저에서 Claude 계정 승인 → 터미널로 복귀 → 입력창 `>` 등장.
- **학습자에게**: "직접 `claude` 쳐서 브라우저 승인까지 완료해보세요. 내 파일이 어디 올라가지 않으니 안심하세요"

---

**[스텝 4] 사업 폴더로 이동 후 재기동**
- **시연**: `/exit` → `cd ~/practice-data/creator` → `ls` → `claude`
- **보여줄 것**: `/exit`로 채팅창 나가기 → `cd`로 실습 폴더 이동 → `ls`로 파일 7개 목록 확인(course_sales_monthly.csv, newsletter_stats.csv 등) → 다시 `claude`로 켜기.
- **학습자에게**: "직접 `ls` 쳐서 파일 목록이 뜨는지 확인해보세요"

---

**[스텝 5] 한국어 첫 대화 — 실제 파일 숫자 꺼내오기**
- **시연**:
  ```
  안녕! 나는 1인 콘텐츠 사업자 지우야. 온라인 강의·뉴스레터·컨설팅을 혼자 운영해.
  지금 이 폴더에 있는 파일들을 보고,
  각 파일이 뭔지 한 줄씩, 그리고 내가 지금 할 수 있는 가장 현실적인 일 3가지를 알려줄래?
  README.md부터 읽고 정리해줘.
  ```
  → 응답 확인 후 이어서:
  ```
  course_sales_monthly.csv 를 실제로 열어서,
  매출이 가장 컸던 한 달과 그 강의 이름을 딱 한 줄로 알려줘.
  ```
- **보여줄 것**: AI가 폴더 안 실제 파일 이름을 읽고 답함. 이어 CSV를 직접 열어 실제 숫자(예: 2026-01 "챗GPT로 콘텐츠 10배 빠르게" 1,477만원)를 꺼내오는 장면.
- **학습자에게**: "직접 자기소개를 넣어 첫 질문을 던져보세요. 답이 어려우면 '더 쉽게', '다시' 라고 하면 됩니다"

- 더 깊은 실습·트러블슈팅 전체는 `/lesson-cc-codex cc-01`에서 자습 가능.

## 📌 강사 노트
- **자주 막히는 지점 1 — `claude: command not found` (설치 직후)**: 새로 깐 프로그램이 현재 창에 아직 안 잡힌 것. 터미널을 완전히 껐다 새로 열어 `claude --version` 다시 치면 대부분 해결. Node.js가 없다면 nodejs.org LTS 먼저.
- **자주 막히는 지점 2 — 로그인 브라우저가 안 열림**: 화면에 뜬 긴 주소를 복사해 브라우저에 직접 붙여넣기. 또는 채팅창 안(`>` 상태)에서 `/login` 입력.
