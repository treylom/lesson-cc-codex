# 1-2: Claude Code로 파일 다루기 — 폴더 탐색 → 분석 → 보고서 초안

## 학습 목표
- Claude Code에게 폴더 탐색·파일 읽기를 시킬 수 있다.
- 매출 데이터(csv)와 회의록(md)에서 패턴·이슈를 함께 찾는다.
- 숫자와 현장 맥락을 연결해 **보고서 초안 1개**를 만든다.
- "관찰 → 해석 → 제안" 구조를 익힌다.

## 사전 준비
- 1-1 완료(첫 대화 경험).
- 현재 폴더에 `practice-data/`(sales_data.csv·meeting_notes.md·README.md)가 있는지 확인.
- 터미널에서 Claude Code 실행 상태.

## 진행

### 도입 (3분)
- 오늘은 AI에게 **내 폴더의 실제 파일**을 읽히고 일을 시킨다.
- 비유: 새 동료에게 "이 서랍 안에 뭐가 있는지 봐주고, 중요한 것만 정리해줘"라고 부탁하는 것.
- 순서: ① 폴더 탐색 → ② csv 패턴 → ③ 회의록 이슈 → ④ 둘 연결 → ⑤ 보고서 초안.

### 실습 1 — 폴더 탐색 (8분)
따라 하기:
1. 폴더에 뭐가 있는지 묻는다.
```text
이 폴더(practice-data 포함)에 어떤 파일이 있고, 무엇부터 보면 좋은지 알려주세요.
```
2. README를 먼저 읽어달라고 한다.
```text
practice-data/README.md 를 읽고, 이 데이터로 제가 뭘 할 수 있는지 3줄로 정리해주세요.
```

기대 결과: 파일 목록 + 각 파일 역할 파악. "어디서부터 볼지"가 보인다.

### 실습 2 — 매출 데이터 패턴 (10분)
따라 하기:
1. csv를 읽고 패턴을 찾게 한다.
```text
practice-data/sales_data.csv 를 읽고, 눈에 띄는 패턴 2~3가지를 비개발자도 알기 쉽게 알려주세요. 숫자 근거도 함께요.
```
2. 한 가지가 궁금하면 파고든다.
```text
7~8월에 support_tickets(문의)와 returns(반품)가 같이 오른 것 같아요. 맞는지, 왜 그럴지 데이터로 짚어주세요.
```

관찰 포인트: AI가 추측만 하면 "csv의 실제 숫자로 근거를 들어주세요"라고 요청. 숫자를 직접 확인하게 만드는 게 핵심.

### 실습 3 — 회의록 이슈 + 연결 (10분)
따라 하기:
1. 회의록에서 반복 이슈를 정리시킨다.
```text
practice-data/meeting_notes.md 에서 반복되는 운영 문제를 정리해주세요.
```
2. 숫자와 맥락을 연결한다.
```text
매출 csv의 7~8월 문의·반품 급증과 회의록 내용을 연결해서, 무슨 일이 있었는지 한 문단으로 설명해주세요.
```

기대 결과: "여름 주문 폭증 + 배송 지연 → 반품·문의 동시 급증" 같은 *숫자+맥락* 해석.

### 실습 4 — 보고서 초안 (12분)
따라 하기:
1. 보고서 구조를 잡는다.
```text
지금까지 찾은 내용을 "관찰 → 해석 → 제안" 순서로 1페이지 보고서 초안으로 만들어주세요. 비개발자 팀장이 읽을 거예요.
```
2. 파일로 저장하게 한다.
```text
이 보고서를 현재 폴더에 보고서.md 로 저장해주세요.
```
3. 검토 후 다듬는다.
```text
'제안' 부분이 추상적이에요. 당장 다음 주에 할 수 있는 구체적 행동 2개로 바꿔주세요.
```

⚠️ **검토 의무**: AI가 만든 보고서를 그대로 제출하지 말고 직접 읽고 사실·숫자를 확인한다.

### 마무리 (4분)
- 오늘 한 사이클: 탐색 → 분석 → 연결 → 보고서 초안 → 검토.
- 한 줄 정리: **"AI는 초안을 빠르게, 검토는 사람이 정확하게."**
- 다음 모듈(2-1)에서는 같은 종류의 일을 **Codex**라는 다른 AI 도구로 해보고, 둘을 비교한다.

## 트러블슈팅
- **파일을 못 읽는다고 함**: 현재 폴더 위치 확인("지금 어느 폴더에 있는지 알려주세요") + 경로를 `practice-data/파일명` 으로 정확히.
- **csv 숫자를 지어내는 느낌**: "csv의 실제 값을 인용해서 다시 확인해주세요"로 근거 요구.
- **보고서가 너무 길다**: "한 페이지·핵심만으로 줄여주세요."
- **저장이 안 됨**: 쓰기 권한/폴더 위치 확인. "보고서.md 가 실제로 만들어졌는지 확인해주세요."
